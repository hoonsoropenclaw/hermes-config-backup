## Description: <br>
Front-end design skill for building React UIs with Ant Design (antd): component patterns, layout, forms, tables, and theming/tokens via ConfigProvider. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[FelipeOFF](https://clawhub.ai/user/FelipeOFF) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and engineers use this skill to build React interfaces with Ant Design v5 patterns for layouts, navigation, forms, tables, modals, drawers, search, uploads, theming, and accessibility. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated front-end code may be adapted to real APIs, forms, tables, or upload flows without sufficient review. <br>
Mitigation: Review generated application code, validation behavior, API integrations, and upload handling before using it with production data or endpoints. <br>
